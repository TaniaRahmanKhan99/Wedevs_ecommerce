# 🧪 Playwright Automation Framework (POM + JavaScript)

This project is an end-to-end UI automation framework built using **Playwright**, **JavaScript**, and **Page Object Model (POM)** design pattern.

It automates the user flow of:
- User Registration
- User Login
- Profile Page Verification

---

# 🚀 Tech Stack

- Playwright
- JavaScript (ES6)
- Page Object Model (POM)
- Node.js
- Git & GitHub
